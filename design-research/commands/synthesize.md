# /synthesize

Synthesize qualitative research data into structured insights.

## Steps

1. Use the `affinity-diagram` skill to organise raw data from $ARGUMENTS into themed clusters with insight statements.
2. Use the `jobs-to-be-done` skill to map user motivations and outcome expectations.
3. Combine findings into a prioritised list of design implications and opportunities.

## Output

Affinity diagram with themed clusters, jobs-to-be-done map, and a prioritised set of design implications.

Consider following up with `/design-research:discover` to create personas from these insights, or `/design-research:test-plan` to validate findings with usability testing.
