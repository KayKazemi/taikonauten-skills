# Motion Design

Design meaningful motion — from principles and brand language to scroll animation, transitions, and loading patterns.

**6 skills** and **3 commands** for motion design in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [motion-principles](./skills/motion-principles) | Core principles of motion: timing, easing, choreography, and purpose. |
| [scroll-animation](./skills/scroll-animation) | Design scroll-driven animations and parallax effects. |
| [transition-system](./skills/transition-system) | Define page and component transition systems. |
| [hero-animation](./skills/hero-animation) | Design hero section and entry animations. |
| [brand-motion](./skills/brand-motion) | Create a brand motion language and motion guidelines. |
| [loading-animation](./skills/loading-animation) | Design loading, skeleton, and progress animation patterns. |

## Commands

| Command | Description |
|---------|-------------|
| `/motion-design:design-motion` | Design a complete motion system for a product. |
| `/motion-design:motion-audit` | Audit existing motion for consistency and performance impact. |
| `/motion-design:animate-transition` | Design transitions between specific states or pages. |
