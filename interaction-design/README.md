# Interaction Design

Design meaningful interactions with micro-animations, state machines, gestures, error handling, and feedback patterns.

**7 skills** and **3 commands** for interaction design in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [micro-interaction-spec](./skills/micro-interaction-spec) | Specify micro-interactions with trigger, rules, feedback, and loop/mode definitions. |
| [animation-principles](./skills/animation-principles) | Apply animation principles to UI motion for purposeful, polished component interactions. |
| [state-machine](./skills/state-machine) | Model complex UI behaviour as finite state machines with states and transitions. |
| [gesture-patterns](./skills/gesture-patterns) | Design gesture-based interactions for touch and pointer devices. |
| [error-handling-ux](./skills/error-handling-ux) | Design error prevention, detection, and recovery experiences. |
| [loading-states](./skills/loading-states) | Design loading, skeleton, and progressive content reveal patterns. |
| [feedback-patterns](./skills/feedback-patterns) | Design system feedback for user actions including confirmations and status. |

## Commands

| Command | Description |
|---------|-------------|
| `/interaction-design:design-interaction` | Design a complete interaction flow for a feature or component. |
| `/interaction-design:map-states` | Model the states and transitions for a complex UI component. |
| `/interaction-design:error-flow` | Design a complete error handling flow for a feature. |
