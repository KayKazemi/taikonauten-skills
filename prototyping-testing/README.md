# Prototyping & Testing

Plan and execute design validation through prototyping strategies, usability testing, heuristic evaluation, and A/B experiments.

**8 skills** and **4 commands** for prototyping and testing in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [prototype-strategy](./skills/prototype-strategy) | Choose the right prototyping fidelity and method for the design question. |
| [test-scenario](./skills/test-scenario) | Write usability test scenarios with tasks, success criteria, and observation guides. |
| [heuristic-evaluation](./skills/heuristic-evaluation) | Conduct expert heuristic evaluations using Nielsen's heuristics and domain-specific criteria. |
| [a-b-test-design](./skills/a-b-test-design) | Design rigorous A/B tests with hypotheses, variants, metrics, and sample size calculations. |
| [user-flow-diagram](./skills/user-flow-diagram) | Create user flow diagrams showing paths, decisions, and branch logic. |
| [wireframe-spec](./skills/wireframe-spec) | Specify wireframe layouts with content priority, component placement, and annotation. |
| [click-test-plan](./skills/click-test-plan) | Design click/first-click tests to evaluate navigation and information findability. |
| [accessibility-test-plan](./skills/accessibility-test-plan) | Create accessibility testing plans covering assistive technologies and WCAG criteria. |

## Commands

| Command | Description |
|---------|-------------|
| `/prototyping-testing:prototype-plan` | Create a prototyping and testing plan for a design initiative. |
| `/prototyping-testing:evaluate` | Run a heuristic evaluation of an existing design. |
| `/prototyping-testing:test-plan` | Design a complete usability testing plan. |
| `/prototyping-testing:experiment` | Design an A/B experiment for a design hypothesis. |
