# Client & Pitch

Win and manage client work — from the first brief to the final deliverable.

**6 skills** and **3 commands** for agency client work in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [proposal-writing](./skills/proposal-writing) | Structure and write compelling client proposals. |
| [project-scoping](./skills/project-scoping) | Define scope, deliverables, timeline, and budget. |
| [client-brief](./skills/client-brief) | Extract and formalize client briefs from conversations. |
| [kickoff-facilitation](./skills/kickoff-facilitation) | Plan and run effective project kickoff meetings. |
| [pitch-structure](./skills/pitch-structure) | Structure a winning pitch presentation for new business. |
| [feedback-management](./skills/feedback-management) | Manage client feedback rounds and revision workflows. |

## Commands

| Command | Description |
|---------|-------------|
| `/client-pitch:write-proposal` | Write a complete client proposal from a brief. |
| `/client-pitch:run-kickoff` | Prepare and facilitate a project kickoff. |
| `/client-pitch:scope-project` | Define and document full project scope. |
