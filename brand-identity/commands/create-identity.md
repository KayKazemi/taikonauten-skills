# /create-identity

Build a complete brand identity from strategy to visual system.

## Steps

1. **Brand foundation**: Use the `brand-strategy` skill to define or confirm purpose, values, positioning, and personality for $ARGUMENTS.
2. **Color system**: Use the `color-identity` skill to build a brand palette tied to personality — primary, secondary, neutral, and semantic roles with accessibility checks.
3. **Typography**: Use the `typography-identity` skill to select typefaces and define the full typographic hierarchy.
4. **Logo guidance**: Use the `logo-system` skill to define required logo variants, clear space rules, and usage guidance.
5. **Visual identity system**: Use the `visual-identity` skill to define imagery style, graphic language, and layout principles.
6. **Compile guidelines**: Use the `brand-guidelines` skill to structure all outputs into a coherent brand guidelines document.

## Output

Complete brand identity package: strategy summary, color system, type system, logo usage rules, visual identity principles, and a structured guidelines document.

Consider following up with `/brand-identity:define-voice` to complete the brand with a communication layer.
