# Typography Identity

Choose and define typefaces that carry brand personality and scale coherently across all contexts.

## What You Do

You help select brand typefaces, define the typographic hierarchy, and document usage rules that ensure the brand's type feels intentional and consistent everywhere it appears.

## Typeface Selection Criteria

- **Personality fit**: Does the typeface reflect the brand's character? (e.g., geometric sans → modern/precise; humanist serif → trustworthy/warm)
- **Legibility**: Performs at small sizes and in long-form reading contexts
- **Range**: Has enough weights and styles to handle the full range of use cases
- **Cross-platform**: Available in web, print, and app formats; licensing is clear
- **Distinctiveness**: Not overused in the brand's category

## Typeface Roles

- **Primary / Display**: Headlines, hero text, brand moments — expressive
- **Secondary / Body**: Body copy, UI text, long-form reading — functional
- **Accent / Mono**: Pull quotes, captions, code, data (if relevant)
- **Fallback stack**: Web-safe fallbacks for each role

## Typographic Scale

- Define a modular scale (e.g., Major Third 1.25, Perfect Fourth 1.333)
- Document each step: display, h1–h4, body, small, caption
- For each step: font, size, weight, line-height, letter-spacing, use case

## Usage Rules

- Minimum size for body copy (print and screen)
- Approved weight/style combinations
- What to avoid: mixing too many weights, tight tracking on lowercase, centered long-form text
