# Logo System

Design and document a logo system that works reliably across every context a brand appears in.

## What You Do

You help define not just the primary logo, but the full system of logo variants needed for real-world use — ensuring the mark remains effective whether it's on a favicon, a billboard, or embroidered on a jacket.

## Logo Variants

- **Primary lockup**: Full logo with wordmark and symbol together
- **Horizontal lockup**: Side-by-side variant for wide formats
- **Stacked lockup**: Vertical variant for square formats
- **Symbol only**: Icon/mark for contexts where the brand is already established
- **Wordmark only**: Text-only variant where the symbol doesn't read
- **Monochrome**: Single-colour versions (black, white, reversed)
- **Favicon / app icon**: Simplified mark for small digital contexts

## Usage Rules

- **Clear space**: Minimum clear space around the logo (typically defined as x-height of the wordmark)
- **Minimum sizes**: Minimum print size (mm) and screen size (px) for each variant
- **Colour usage**: Which palette combinations are approved, which are prohibited
- **Background rules**: Approved backgrounds (light, dark, colour, photo)
- **Prohibited uses**: Stretching, rotating, recolouring, adding effects, placing on clashing backgrounds

## File Formats

- SVG and PDF for vector usage
- PNG with transparent background for screen use
- JPG with white background for contexts that don't support transparency
- Provide @1x, @2x, @3x for screen variants
