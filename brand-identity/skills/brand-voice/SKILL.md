# Brand Voice

Define how a brand speaks so that every piece of communication — from error messages to ad headlines — sounds like it comes from the same place.

## Context

You are a brand language strategist helping define the voice for $ARGUMENTS. If existing communication samples are provided, analyse them first for patterns.

## Domain Context

- Voice is consistent (who the brand is); tone shifts by context (how it speaks in different situations).
- Nielsen Norman Group: A distinct voice builds trust and memorability; an inconsistent one erodes both.
- Good voice guidelines give writers permission, not a straitjacket.

## Voice Framework

1. **Voice pillars**: 3–4 core characteristics (e.g., "Direct", "Warm", "Curious")
   - For each: a one-line definition, 2–3 writing examples, and what it is not
2. **Tone spectrum**: How the voice shifts across contexts (e.g., formal → informal, serious → playful)
   - Map tone for key contexts: error states, marketing, onboarding, support, social
3. **Vocabulary**
   - Words and phrases the brand uses
   - Words and phrases to avoid
   - Technical language: how much jargon is appropriate and for which audience
4. **Messaging hierarchy**
   - Brand tagline or signature line
   - Elevator pitch (1 sentence, 1 paragraph, 1 page)
   - Key messages by audience segment
5. **Writing principles**
   - Sentence length and rhythm preferences
   - Active vs. passive voice
   - Punctuation and capitalisation quirks

## Instructions

Work through each section of the voice framework for $ARGUMENTS. For each voice pillar, write three before/after examples that demonstrate the difference between on-brand and off-brand writing.
