# Visual Identity

Translate brand strategy into a coherent, scalable visual language.

## What You Do

You help design and document visual identity systems — the set of visual elements that make a brand immediately recognisable and consistently expressed across all touchpoints.

## Identity System Components

- **Logo**: Primary mark, wordmark, symbol, and combinations
- **Color**: Primary palette, secondary palette, tints, semantic roles
- **Typography**: Brand typefaces, scale, hierarchy
- **Imagery style**: Photography art direction, illustration style, iconography
- **Graphic language**: Patterns, shapes, textures, compositional rules
- **Motion language**: How the brand moves (see `brand-motion`)
- **Space and layout**: White space philosophy, grid approach, density

## Design Principles

- **Distinctiveness**: The system should be recognisable without a logo present
- **Flexibility**: Must work across digital, print, environmental, and social contexts
- **Coherence**: All elements should feel like they come from the same world
- **Scalability**: Works at business card size and billboard size
- **Accessibility**: Contrast ratios and legibility must meet WCAG 2.2 AA as a minimum

## Process

1. Audit existing identity (if applicable) for consistency gaps and equity worth retaining
2. Translate brand personality adjectives into visual equivalents
3. Define each system component with rationale tied back to brand strategy
4. Show the system in use across 3–5 representative touchpoints
5. Document usage rules and anti-patterns for each element
