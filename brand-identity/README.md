# Brand Identity

Build and articulate brand identities — from strategy and positioning to visual systems, voice, and guidelines.

**7 skills** and **3 commands** for brand identity work in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [brand-strategy](./skills/brand-strategy) | Define brand positioning, purpose, values, and audience. |
| [visual-identity](./skills/visual-identity) | Design cohesive visual identity systems. |
| [logo-system](./skills/logo-system) | Define logo variations, usage rules, and clear space. |
| [brand-guidelines](./skills/brand-guidelines) | Create comprehensive brand guidelines documents. |
| [brand-voice](./skills/brand-voice) | Define tone of voice and messaging frameworks. |
| [color-identity](./skills/color-identity) | Build brand color systems with semantic and expressive roles. |
| [typography-identity](./skills/typography-identity) | Select and define brand typefaces and typographic hierarchy. |

## Commands

| Command | Description |
|---------|-------------|
| `/brand-identity:brand-audit` | Audit an existing brand for consistency and strategic clarity. |
| `/brand-identity:create-identity` | Create a complete visual identity from brand strategy. |
| `/brand-identity:define-voice` | Define brand voice, tone, and messaging framework. |
