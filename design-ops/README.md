# Design Ops

Streamline design operations with critique frameworks, handoff specs, sprint planning, review processes, and team workflows.

**9 skills** and **3 commands** for design operations in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [design-critique](./skills/design-critique) | Facilitate structured design critiques with clear feedback frameworks and actionable outcomes. |
| [handoff-spec](./skills/handoff-spec) | Create developer handoff specifications with measurements, behaviours, assets, and edge cases. |
| [design-sprint-plan](./skills/design-sprint-plan) | Plan and facilitate design sprints from challenge framing through prototype testing. |
| [design-review-process](./skills/design-review-process) | Establish design review gates with criteria, checklists, and approval workflows. |
| [version-control-strategy](./skills/version-control-strategy) | Define version control strategies for design files, components, and libraries. |
| [design-qa-checklist](./skills/design-qa-checklist) | Create QA checklists for verifying design implementation accuracy. |
| [team-workflow](./skills/team-workflow) | Design team workflows covering task management, collaboration rituals, and tooling. |
| [client-feedback-management](./skills/client-feedback-management) | Structure and manage client feedback cycles to keep reviews productive and revisions within scope. |
| [revision-workflow](./skills/revision-workflow) | Define a clear, trackable revision workflow with version control and change request processes. |

## Commands

| Command | Description |
|---------|-------------|
| `/design-ops:plan-sprint` | Plan a design sprint for a specific challenge. |
| `/design-ops:handoff` | Generate a developer handoff package for a design. |
| `/design-ops:setup-workflow` | Set up a design team workflow and rituals. |
