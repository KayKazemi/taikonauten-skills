# Designer Toolkit

Essential utilities for writing rationale, building presentations, crafting pitch decks, case studies, UX writing, and driving adoption.

**7 skills** and **4 commands** for designer productivity in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [design-rationale](./skills/design-rationale) | Write clear design rationale connecting decisions to user needs, business goals, and principles. |
| [presentation-deck](./skills/presentation-deck) | Structure compelling design presentations for stakeholders, reviews, and showcases. |
| [pitch-deck](./skills/pitch-deck) | Structure and write a pitch deck for new business, investors, or internal stakeholders. |
| [case-study](./skills/case-study) | Craft portfolio-ready case studies that tell the story of a design project. |
| [design-token-audit](./skills/design-token-audit) | Audit design token usage across a product for consistency and coverage. |
| [ux-writing](./skills/ux-writing) | Write effective UI copy including microcopy, error messages, empty states, and CTAs. |
| [design-system-adoption](./skills/design-system-adoption) | Create adoption strategies and materials to drive design system usage across teams. |

## Commands

| Command | Description |
|---------|-------------|
| `/designer-toolkit:write-rationale` | Write design rationale for a set of design decisions. |
| `/designer-toolkit:build-presentation` | Structure a design presentation for a specific audience. |
| `/designer-toolkit:write-case-study` | Create a portfolio case study from a project summary. |
| `/designer-toolkit:write-pitch` | Build a complete pitch deck for new business or a design proposal. |
