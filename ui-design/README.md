# UI Design

Craft polished user interfaces with layout grids, color systems, typography scales, responsive patterns, and visual hierarchy.

**9 skills** and **4 commands** for UI design in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [layout-grid](./skills/layout-grid) | Define responsive layout grid systems with columns, gutters, margins, and breakpoints. |
| [color-system](./skills/color-system) | Build a comprehensive color system with palette generation and accessibility compliance. |
| [typography-scale](./skills/typography-scale) | Create a modular typography scale with size, weight, and line-height relationships. |
| [responsive-design](./skills/responsive-design) | Design adaptive layouts and interactions across all screen sizes and inputs. |
| [visual-hierarchy](./skills/visual-hierarchy) | Establish clear visual hierarchy through size, weight, color, spacing, and position. |
| [spacing-system](./skills/spacing-system) | Create a consistent spacing system based on a base unit with application rules. |
| [dark-mode-design](./skills/dark-mode-design) | Design effective dark mode interfaces with proper color adaptation and contrast. |
| [illustration-style](./skills/illustration-style) | Define an illustration style guide with visual language and usage rules. |
| [data-visualization](./skills/data-visualization) | Design clear, accessible data visualisations with appropriate chart selection. |

## Commands

| Command | Description |
|---------|-------------|
| `/ui-design:design-screen` | Design a complete screen layout from a description or requirements. |
| `/ui-design:color-palette` | Generate a full color palette with semantic mapping and accessibility checks. |
| `/ui-design:type-system` | Create a complete typography system from brand fonts or requirements. |
| `/ui-design:responsive-audit` | Audit a design for responsive behaviour across breakpoints. |
