# UX Strategy

Shape product direction through competitive analysis, market positioning, design principles, experience mapping, and strategic alignment.

**9 skills** and **3 commands** for UX strategy in [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|-------|-------------|
| [competitive-analysis](./skills/competitive-analysis) | Conduct a structured competitive analysis comparing UX patterns, features, strengths, and gaps. |
| [design-principles](./skills/design-principles) | Define actionable design principles that guide decision-making and resolve trade-offs. |
| [experience-map](./skills/experience-map) | Create a holistic experience map of user touchpoints, channels, and relationships. |
| [north-star-vision](./skills/north-star-vision) | Articulate a compelling north-star product vision that aligns teams. |
| [opportunity-framework](./skills/opportunity-framework) | Identify, evaluate, and prioritise design opportunities using impact-effort frameworks. |
| [design-brief](./skills/design-brief) | Write a comprehensive design brief defining problem space, constraints, and success criteria. |
| [stakeholder-alignment](./skills/stakeholder-alignment) | Create alignment artefacts including RACI matrices and decision frameworks. |
| [metrics-definition](./skills/metrics-definition) | Define UX metrics and KPIs connecting design decisions to measurable outcomes. |
| [market-positioning](./skills/market-positioning) | Define where a product sits in its market relative to competitors and articulate the positioning clearly. |

## Commands

| Command | Description |
|---------|-------------|
| `/ux-strategy:strategize` | Develop a complete UX strategy for a product or feature area. |
| `/ux-strategy:benchmark` | Run a competitive benchmarking analysis across products. |
| `/ux-strategy:frame-problem` | Structure an ambiguous challenge into a clear problem definition. |
